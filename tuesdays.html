Tuesdays

Digital Fashion Magazine

Project in SPD Milan - Digital Typography

Web design Program

TUESDAYS is a Digital Magazine that targets a young audience who loves fashion and consumes it through smartphones and the web. I used a dynamic sans-serif typography for the logo, one of the type trends that attracts young people. It seems modern but it’s a retro type (nostalgic), it’s been around for a very long time like Fashion. The evolution of Fashion is a cycle where modern designers bring back the old styles.
